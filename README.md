# AI Studio 对话管理器

![Version](https://img.shields.io/badge/version-1.1.0-blue?style=flat-square)
![Platform](https://img.shields.io/badge/platform-Chrome%20%7C%20Edge-green?style=flat-square)
![License](https://img.shields.io/badge/license-MIT-orange?style=flat-square)

一个专为 [Google AI Studio](https://aistudio.google.com/) 打造的浏览器增强扩展。

在进行长文本对话或 Prompt 调试时，它可以为您提供一个直观的**悬浮管理面板**。通过纯净的对话目录、高效的搜索过滤以及强大的批量管理功能，让您在几十轮的复杂对话中也能游刃有余。

---

## ✨ 主要功能

### 🎈 灵动悬浮球 & 侧边栏
*   **无感常驻**：默认在页面右下角显示一个小圆球，不占用屏幕空间。
*   **物理手感**：支持随意拖拽，松手后自动吸附至屏幕边缘，丝滑流畅。
*   **侧边关闭**：为了防止误触，我们在面板最左侧设计了独特的**侧边触控条**。只需点击面板左侧突出的“耳朵”区域，或直接按 `ESC` 键，即可快速收起面板。

### � 实时搜索与过滤
*   **即时筛选**：在几十轮的对话中迷路了？直接输入关键词（支持空格分隔多词），列表会立刻过滤显示匹配的卡片。
*   **高亮标记**：匹配的关键词会以**黄色高亮**显示，帮您瞬间定位到目标 Prompt。
*   **智能屏蔽**：自动过滤掉 AI 的元数据（如 Token 统计、文件名等噪音），只保留最核心的对话内容。

### �️ 高效多选交互
我们提供了操作系统级的多选体验，让批量管理变得前所未有的简单：
*   **Shift + 点击 (范围选择)**：点击第一条，按住 `Shift` 再点击第十条，中间所有对话瞬间选中。
*   **Alt + 拖拽 (画笔刷选)**：按住 `Alt` 键，用鼠标像画笔一样扫过列表，所到之处自动勾选。

### 🗑️ 智能批量删除
*   **连坐清理**：勾选一个用户提问，插件会自动关联并删除该提问引发的所有后续内容（包括 AI 回复、思考过程、错误提示等）。
*   **安全可靠**：针对 Angular 动态网页深度优化，确保删除操作稳定、彻底，不留残余。

### ⭐ 标注与导航
*   **重点星标**：点击星星即可收藏重要对话节点，刷新页面不丢失。
*   **快捷跳转**：点击列表项可直接滚动至对应对话位置并高亮提醒。
*   **一键到底**：内置悬浮按钮，一键直达对话流最底部。
*   **深色模式**：完美适配 Google AI Studio 的深色/浅色主题。

## 📥 安装指南

本工具为浏览器扩展程序（Extension），适用于 Chrome、Edge 等基于 Chromium 内核的浏览器。

### 第 1 步：下载源码
1.  下载本项目的 ZIP 压缩包或 Clone 到本地。
2.  解压到一个固定文件夹（安装后请勿删除）。

### 第 2 步：开启开发者模式
1.  在浏览器地址栏输入 `chrome://extensions/` (Chrome) 或 `edge://extensions/` (Edge)。
2.  打开右上角的 **“开发者模式” (Developer mode)** 开关。

### 第 3 步：加载扩展
1.  点击左上角的 **“加载已解压的扩展程序” (Load unpacked)**。
2.  选择解压后的文件夹（包含 `manifest.json` 的那一层）。
3.  🎉 安装完成！刷新 AI Studio 页面即可看到右下角的悬浮球。

## ⚠️ 注意事项

*   **数据安全**：本插件所有数据（如星标状态）均存储在您本地的浏览器中 (`chrome.storage.local`)，绝不上传至任何服务器。
*   **操作安全**：批量删除功能虽然强大且经过测试，但**删除操作不可逆**。建议在执行大批量删除前仔细确认选中内容。
